Este projeto tem como objetivo armazenar os fontes da empresa TreLac e forma de gerenciar versionamentos e prover um armazenamento seguro.
Para permitir que outros analistas criem, modifiquem ou até removam arquivos do projeto, basta adicionar seus respectivos e-mails aos colaboradores deste projeto por meio do Git-hub
Os colaboradores podem realizar um Clone do projeto em seus terminais e realizarem as modificações necessárias, posteriormente, devem publicar suas versões e assim, todo o rastreio dos fontes será mantido.
